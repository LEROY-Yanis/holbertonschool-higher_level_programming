# My JavaScript Project

This project contains a series of JavaScript scripts that demonstrate various programming concepts and techniques. Each script is located in the `src` directory and serves a specific purpose.

## File Descriptions

- **1-multi_languages.js**: Prints three predefined lines of text.
- **2-arguments.js**: Checks the number of arguments passed and prints a corresponding message.
- **3-value_argument.js**: Prints the first argument passed or a message if no arguments are provided.
- **4-concat.js**: Prints two arguments in a specific format.
- **5-to_integer.js**: Attempts to convert the first argument to an integer and prints it or a message if it cannot be converted.
- **6-multi_languages_loop.js**: Prints three lines using an array and a loop.
- **7-multi_c.js**: Prints "C is fun" a specified number of times based on the first argument.
- **8-square.js**: Prints a square of a specified size using the character 'X'.
- **9-add.js**: Defines a function to add two integers and prints the result.
- **10-factorial.js**: Computes and prints the factorial of a given integer recursively.
- **11-second_biggest.js**: Finds and prints the second biggest integer from the list of arguments.
- **12-object.js**: Modifies an object by changing one of its properties.
- **13-add.js**: Contains a function that returns the addition of two integers.

## Installation

To install the necessary dependencies, run:

```
npm install
```

## Usage

Each script can be executed using Node.js. For example, to run `1-multi_languages.js`, use the following command:

```
node src/1-multi_languages.js
```
