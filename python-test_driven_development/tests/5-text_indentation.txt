The ``5-text_indentation`` module
==================================

Using ``text_indentation``
--------------------------

Import the function:

    >>> text_indentation = __import__('5-text_indentation').text_indentation

Test with simple sentence:

    >>> text_indentation("Hello.")
    Hello.
    <BLANKLINE>

Test with question:

    >>> text_indentation("Hello?")
    Hello?
    <BLANKLINE>

Test with colon:

    >>> text_indentation("Hello:")
    Hello:
    <BLANKLINE>

    >>> text_indentation(123)
