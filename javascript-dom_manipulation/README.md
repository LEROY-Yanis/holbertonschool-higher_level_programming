# JavaScript - DOM Manipulation

This project contains JavaScript exercises focused on DOM (Document Object Model) manipulation.

## Files

- `0-script.js` - Changes header text color to red
- `1-script.js` - Changes header color on click
- `2-script.js` - Adds a CSS class on click
- `3-script.js` - Toggles classes on click
- `4-script.js` - Adds list items dynamically
- `5-script.js` - Updates header text on click
- `6-script.js` - Fetches and displays Star Wars character
- `7-script.js` - Fetches and lists Star Wars movies
- `8-script.js` - Fetches and displays greeting in French
